ajax的严格模式，https页面会拦截非https的ajax请求.

基于[ajah](https://github.com/axetroy/ajah)实现

## Installation
```bash
npm install ajax-strict-mode
```

## Usage

```javascript
import ajaxStrictMode from 'ajax-strict-mode';
ajaxStrictMode()
```
